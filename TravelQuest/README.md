# TravelQuest - Frontend

Aceasta este partea de frontend pentru aplicatia TravelQuest.  
Frontend-ul este construit folosind React + Vite si se ocupa de afisarea interfetei grafice si interactiunea cu utilizatorul. Acesta va comunica cu backend-ul folosind Axios, prin endpoint-uri REST.

---

## 1. Tehnologii folosite

- React
- Vite (sistem de build si dev server)
- Axios (pentru cereri HTTP)
- React Router (pentru navigatie)
- Vitest + Testing Library (pentru teste unitare)

---

## 2. Intructiuni pentru a rula proiectul

Inainte de a incepe, trebuie instalate:

- Node.js (versiunea 18 sau mai noua)
- npm (versiunea 9 sau mai noua)

Dupa clonarea repository-ului, din /frontend trebuie rulate:

* instalarea dependintelor:

```
npm install
```

* rularea proiectului:

``` 
npm run dev
```

Aplicatia va fi accesibila la ``` http://localhost:5173 ```
